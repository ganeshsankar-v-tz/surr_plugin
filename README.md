# surr_plugin

A new Flutter project.

## Permissions

This plugin requires the following permissions on Android:

*   `android.permission.RECORD_AUDIO`: To record audio via the device microphone or external devices.
*   `android.permission.USB_PERMISSION`: To access external USB recording devices.

Make sure to request `Permission.microphone` at runtime before calling `startRecorder`.
